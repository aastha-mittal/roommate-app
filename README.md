# Roommate App

A shared living app for roommates—chore tracking, expense splitting, and household coordination in one place.

## Features (planned)

- **Chore tracking** — Assign and rotate tasks (cleaning, trash, etc.)
- **Expense splitting** — Log shared bills and split costs fairly
- **Household calendar** — Events, move-in/move-out, guest plans
- **Notes & announcements** — House rules, WiFi, emergency contacts
